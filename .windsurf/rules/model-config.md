# Model Configuration Guide

## Препоръчани модели (от най-добър към добър)

### 1. SWE-1.5 (DEFAULT — БЕЗПЛАТЕН)
- **Статус**: Безплатен за всички потребители
- **Скорост**: ~950 tokens/sec (13x по-бързо от Claude Sonnet 4.5)
- **Качество**: Близко до frontier модели
- **Препоръка**: ИЗПОЛЗВАЙ ТОЗИ — най-добрият безплатен вариант

### 2. SWE-1 Lite (Безплатен)
- За по-прости задачи и autocomplete
- По-лек, по-бърз за рутинни операции

### 3. DeepSeek V3 (BYOK — много евтин)
- Добавя се чрез Settings → Provider → Custom Model (OpenAI Compatible)
- Качество близко до Claude 3.5 Sonnet
- 64K–128K контекст
- Цена: ~$0.14/$0.28 per 1M tokens (input/output)

## Модели които НЕ работят добре с тази задача
- Kimi K2.5 (free) — недостатъчен за сложна класификация
- Безплатни Llama варианти — твърде слаби за multi-file reasoning

## Как да смениш модела в Cascade
1. Отвори Cascade чата
2. Кликни на модел dropdown (горе вляво)
3. Избери SWE-1.5 (или друг от списъка)
