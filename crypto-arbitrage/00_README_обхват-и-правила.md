# Обхват и правила — Крипто арбитраж

> Този файл описва целите, обхвата и основните правила за секцията за крипто арбитраж.

## Обхват
- CEX ↔ CEX арбитраж (спот)
- CEX ↔ DEX арбитраж
- Funding rate арбитраж
- Триъгълен арбитраж
- Cross-chain арбитраж

## Правила
- [ ] Всички стратегии са **само за учебни цели**
- [ ] Минимална проверка преди прилагане: такси, ликвидност, слиплидж
- [ ] Задължително валидиране с реални данни преди реални средства
- [ ] Документирай всички допускания и рискове
