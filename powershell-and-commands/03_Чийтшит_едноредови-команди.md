# Чийтшит — едноредови команди

> Кратки едноредови команди с обяснение за бърза справка.

| # | Команда | Описание |
|---|---------|----------|
| 1 | `Get-Process \| Sort-Object CPU -Descending \| Select -First 10` | Топ 10 процеси по CPU |
| 2 | `Get-ChildItem -Recurse -File \| Measure-Object -Property Length -Sum` | Общ размер на файловете |
| 3 | `Test-NetConnection -ComputerName google.com -Port 443` | Проверка на HTTPS свързаност |
